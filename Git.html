改变Git diff

  • 建立一个用于DIFF命令，如下/usr/bin/gitdiff 
    #!/bin/bash
vimdiff "$2" "$5"
  • 向gitdiff增加运行权限 
        chmod +x /usr/bin/gitdiff
  • 修改~/.gitconfig文件，增加如下内容 
        [diff]
        extenal = gitdiff
    [pager]
        diff =
    或 
        [diff]
        extenal = gitdiff
    [core]
        pager =

给Git增加别名

    git config alias.co checkout
