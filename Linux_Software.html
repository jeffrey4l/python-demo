Linux软件

  • Ekiga
    • Ekiga(原名 GnomeMeeting )是一款开源的基于GNOME桌面系统的VOIP和电视会议系统。同时支持H.323和SIP协议。它支持多种音频视频解码器，并且可以和其他的支持SIP协议的软件通信，如微软的NetMeeting。
  • Lookit
    • Lookit 特色在于只需一键操作就可以进行快速截图及分享，通过它可以把截图快速上传到图片分享网站 Imgur 或自定义的 FTP 及 SSH 主机上，并且上传完会自动把图片地址存入剪贴板中。 
      Beta 版
	sudo add-apt-repository ppa:lookit/testing 
	sudo apt-get update 
	sudo apt-get install lookit
稳定版
	sudo add-apt-repository ppa:lookit/ppa 
	sudo apt-get update 
	sudo apt-get install lookit
