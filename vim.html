编译

Vim7.3已经出来了好长时间,但是ubuntu的源里一直没有把他加进来,实在引不了了,动手自己编译了一个版本。前段时间编译时一直不能加入python的支持，前两天重新编译时，终于搞定了，在此记录下。

  • 下载vim7.3源代码

下载方式有以下两种：

  • 直接到官方网站上下载
  • 使用Mercurial直接从代码库中下载 
        hg clone https://vim.googlecode.com/hg/ vim
    推荐大家使用第二种方法。
  • 安装编译工具 
        sudo apt-get install build-essential
  • 安装必要的信赖包

    • 在ubuntu下要先安装libncurses5-dev,python2.6-dev这两个包，命令如下： 

        sudo apt-get install libncurses5-dev python2.6-dev
    • 编译代码

    进入vim/src目录，依次运行如下命令 

    ./configure --with-features=huge --enable-pythoninterp=yes --enable-gui --enable-cscope --enable-fontset --enable-perlinterp --enable-rubyinterp --with-python-config-dir=/usr/lib/python2.6/config 
make 
sudo make install

    就可以完成vim7.3的编译安装了。 对于vim7.3的新特性功能，大家可以访问善用佳软所写的文章。 另面，我在github上共享了我的vim配置文件。欢迎大家围观。
